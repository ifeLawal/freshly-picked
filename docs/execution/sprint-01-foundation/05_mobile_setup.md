# Mobile App Setup

## Overview
This step initializes the Expo app and installs all core dependencies the mobile client requires. The goal is a running app with tab navigation in place and every foundational library installed and confirmed working — so all subsequent mobile screens and features have a stable base to build on.

## Requirements
- Expo app created and runnable on simulator/device
- React Navigation installed and configured for tab-based navigation
- React Query installed for server data fetching
- Zustand installed for UI state management
- AsyncStorage installed for local persistence
- Expo AV installed for audio playback support
- Base tab navigation structure in place (Home, Episodes, Search, Lists, Settings)
